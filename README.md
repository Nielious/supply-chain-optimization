# Supply Chain Optimizer

Retail discount optimization using Python, Pandas, and Linear Programming.

## Problem
Analysis of Superstore Sales dataset (9,994 transactions) revealed that Furniture category generates minimal profit despite high sales volume. Tables and Bookcases consistently operate at a loss due to excessive discounting.

## Solution
Built a Linear Programming model using PuLP to find optimal discount levels for each Furniture sub-category while maintaining positive profit margins.

## Results
- Identified that Tables lose 14.8% margin at current 26% discount
- Optimized discounts across 4 sub-categories
- Projected profit improvement: $120,347

## Tech Stack
- Python, Pandas, Matplotlib
- PuLP (Linear Programming)
- Streamlit
